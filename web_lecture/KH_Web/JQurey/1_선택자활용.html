선택자 활용하기

JQuery에서는 element를 불러올 때 선택자를 활용한다.
$('선택자') -> Elemnet
JQuery로 불러온 선택자는 배열형식으로 가져오게 된다.

:first :last :even :odd

작성방법
선택자:first -> (선택자로 선택해서 가져온 element는 배열 형식) 선택된 Elemnet 중 0번 인덱스 태그
선택자:last -> 선택된 Elemnet 중 마지막 인덱스 태그
선택자:even -> 선택된 Elemnet 중 짝수 인덱스 태그
선택자:odd -> 선택된 Elemnet 중 홀수 인덱스 태그

:eq , :gt , :lt

  • 선택자:eq(n) -> 선택된 태그중 n번 인덱스 태그를 가져옴
  • 선택자:gt(n) -> 선택된 태그중 n번보다 큰 번호를 가진 인덱스 태그들을 가져옴
  • 선택자:lt(n) -> 선택된 태그중 n번보다 작은 번호를 가진 인덱스 태그들을 가져옴
