생성한 태그를 페이지에 추가하기

jquery에 의해 생성된 태그는 메모리 상에서 객체로 존재함 그 객체를 페이지에 보이게 하려면 페이지에 있는 태그에 추가해줘야한다.

  • 자식객체로 추가하기
    1. 부모태그.append(추가태그) / 추가태그.appendTo(부모태그) : 부모태그의 맨 마지막 자식으로 추가태그를 추가함.
    2. 부모태그.prepand(추가태그) / 추가태그.prepandTo(부모태그) : 부모태그의 맨 위 자식으로 추가태그를 추가함.
  • 형제 객체로 추가하기
    1. 선택태그.before(추가태그) / 추가태그.insertBefore(선택태그) : 선택태그 밑에 추가태그를 추가함
    2. 선택태그.after(추가태그) / 추가태그.insertAfter(선택태그) : 선택태그의 뒤에 추가태그를 추가함
