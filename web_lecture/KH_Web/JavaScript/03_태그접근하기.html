태그 접근하는 방법

js에서 태그는 하나의 객체로 본다 (DOM)
객체에서는 태그를 표현하는 properties(속성)가 있다,
태그에 접근하는 방법은 4가지가 있다.
속성값을 이용해서 태그를 가져온다.

  1. id 값을 기준으로 가져오기
  2. class 값을 기준으로 가져오기
  3. tagName 값을 기준으로 가져오기
  4. name 값을 기준으로 가져오기

id 값을 기준으로 가져오기

태그의 id속성에 있는 값을 기준으로 태그를 가져옴
document.getElementById("id속성값"); -> id 속성이 일치하는 태그
한 개를 가져옴

class 속성을 기준으로 값을 가져오기

태그의 설정된 class 속성 값을 기준으로 태그 객체 가져오기
중복값을 허용 -> 다수가 있으면 다수의 태그 객체를 불러옴
document.getElementByClassName("class 속성값");

선택자를 통해 가져오기

css에서 사용한 선택자를 이용해서 태그를 지정해여 가져오는것
document.querySelector("선택자"); -> 한개의 태그만 가져오는 것, 많으면 첫번째만
document.querySelectorAll("선택자"); -> 선택자에 해당하는 거 다가져오는 것
