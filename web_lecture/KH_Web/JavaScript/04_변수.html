변수 선언하기

변수 선언하기
var+ 예약어; (js6 이후는 let, const 사용한다.)
자바스크립트에서 변수를 선언할 때는 자료형을 지정하지 않는다.

변수는 접근할 수 있는 범위에 따라 영역(scope)이 구분
global scope : page내 어디서든 접근이 가능한 변수-> window객체의 멤버변수
local scope : function 내부에 선언한 변수 -> function 내부에서만 사용가능

자바스크립트는 블록범위를 허용하지 않는다.
자바
if(){ String test} -> test ? 외부접근 불가능
for(){int a} -> a ? 불가능
JS
if(){var test} -> console.log(test) -> 접근가능

변수를 선언시 주의할점
변수명이 중복되면 -> 에러발생 x 덮어씀
es6 이후로 선언하는 방법이 변경됨

ES6에서 변수를 선언하는 예약어를 추가함
블록스코프를 유지하고 중복선언을 방지하는 기능
-> let, const 예약어 사용해서 변수 선언 함
let : 블록스코프영역을 가지는 변수 선언할 때
const : 상수를 저장하는 변수를 선언할 때

자바스크립트 자료형

자바스크립트는 자료형을 가지고 있으며 변수의 자료형이 결정되는 것은
변수에 값(리터럴)을 대입할 때임
각 자료형을 알아보고 리터럴 표현방법을 알아보자
문자열, 숫자, 배열, 객체, 함수, undefined, boolean

변수의 타입 확인하기

typeof : 함수를 이용하면 타입을 알 수 있음
사용법-> typeof 변수명 / typeof(변수명)
