데이터 출력하기

스크립트에서 데이터를 출력하는 방식은 두개로 나눌 수 있음
1. 페이지에 출력하는 방법
2. 브라우저가 출력하는 방법( 알림창, console 창)

  • 페이지에서 출력
    1. document.writer("출력구문 || 태그+출력구문")
    2. 태그선택.innerHTML="출력구문"
  • 브라우저에서 출력
    1. 알림창 출력 : alert("출력문구")
    2. console 창 출력 : console.log("출력문구")

innerHTML 속성 이용하기

innerHTML 속성은 현재 출력하고 있는 문구를 저장하는 속성
innerHTML 속성에 저장되어있는 값이 화면에 출력.
태그.innerHTML="문구";
