자바스크립트 이용하기

자바스크립트로 페이지를 interactive 하게 구성하려면
자바스크립트 구문을 태그내 속성(inline), script 태그내부, 외부 js 파일에 작성한다.

  1. inline: 태그 내부의 이벤트 속성(on)에 직접 코드를 작성함 -> 간단한 것만 사용
  2. internal : 페이지의 script 태그안에 코드를 작성하는 것
  3. external : 특정 확장자(.js) 파일에 js 코드를 작성하고 페이지에서 불러오는 것
    ex) script src="주소" / 서버내부 파일이나 외부주소(https://)에서 지원하는 파일 가능
