인코딩 관련 내장함수

인코딩, 디코딩 관련함수
네트워크상에서 정보(데이터)를 공유하기 위해서 문자에 대한 처리중

인코딩, 디코딩 방법은 mapping되어있다.
ex) a방법으로 인코딩했으면 a방법으로 디코딩해야한다.
인코딩 / 디코딩
escape / unescape : 영문, 숫자, 일부 특수기호(@,*,-,_,.,/) 제외한 모든 문자 인코딩 처리
encodingURI / decodeURI : escape에서 인터넷 주소에 사용되는 일부 특수기호(: ,; ,/ ,= ,&) 빼고 인코딩 처리함
encodeURIComponent / decodeURIComponent : 알파벳, 숫자를 제외한 모든 문자를 인코딩처리
