배열을 선언하고 활용하기

자바스크립트에서 배열은 자바의 ArrayList와 거의 비슷함
1. 다수의 자료형 보관이 가능하다.
2. 길이 제한이 없다.
3. 인덱스로 값을 접근한다.
*DB에서 데이터를 가져온 값, DOM객체를 가져왔을 때 사용

배열 내장함수 이용하기

배열변수.indexOf(매개변수) => 배열에서 매개변수값과 일치하는 요소의 인덱스를 리턴함

배열에 서울 대전 대구 부산 광주 목포 포항 넣고
input 태그로 입력받아 그 지역이 있는지 확인하고 div id="target"에 출력
몇번인덱스에 있을까요?

두개 이상의 배열을 합치는 함수 : concat

배열을 문자열로 합쳐주는 함수 : join()

배열의 순서을 변경해주는 함수 : reverse()

배열의 데이터를 순서정렬하는 함수 : sort()

정렬에 대한 함수를 만들어서 배열을 정렬시킴
-,0,+ 리턴값을 이용해서 처리하게됨 -> Collection.sort();

배열의 데이터를 조작하는 함수

push() : 배열에 맨뒤에 요소를 추가하는 함수
pop() : 배열 맨뒤 인덱스를 제거하고 그걸 반환함.

shitf(): 맨앞에 요소를 제거함
unshift(): 맨앞에 요소를 추가함

slice(): 배열의 일부를 잘라서 새배열로 생성 -> 원본수정 x
slice(시작인덱스 번호, 끝 인덱스 번호)
끝 인덱스 번호는 포함하지 않음(끝 인덱스 번호 빼고 그 앞까지)

splice() : 배열의 특정 위치값을 지우고 그 위치에 값을 추가 -> 원본 수정 o
splice(시작 인덱스, 지울 개수,추가할 값1,추가할 값2....);
추가할 값 안넣으면 -> 원하는 위치 값 삭제
지울 개수 0이면 -> 원하는 위치에 값 넣기

배열에 있는 값을 필터하는 함수

includes(): 특정값이 배열에 있는지 확인하는 함수
find() : 특정기준에 맞는 요소를 찾는 함수 -> 한개만찾음
filter(): 특정기준에 맞는 요소를 찾는 함수 -> 전부 다 찾음

배열 값 순회하는 함수

forEach() : 배열에 있는 모든 값을 접근하는 것
매개변수로 함수를 전달받음, 매개변수는 2개 혹은 3개받음.

누적 계산해주는 함수

reduce(함수[,초기값]) : 특정변수에 return 값을 지속 저장하는 함수
매개변수로 함수를 전달받음 -> 그 함수는 2개의 매개변수를 갖는다.
첫번째 : 초기값 (0번 인덱스 값)
두번째 : 다음값
return 값이 처음 매개변수에 대입됨
