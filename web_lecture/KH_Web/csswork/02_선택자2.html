active : 마우스를 클릭했을 때 style 적용
hober : 마우스가 태그영역에 들어갔을 때 style 적용

문자선택자

::first-letter : 첫번째 글자만 style적용하기
::first-line : 첫번째 줄만 style 적용하기
::after : 태그의 끝문자에 문자 추가
::before : 태그의 시작에 문자 추가
::selection : 마우스 드래그로 선택했을 때 style 적용

구조 선택자

선택자:first-child : 구조상 첫번째인 자식인 선택자 태그
선택자:last-child : 구조상 마지막 자식인 선택자 태그
선택자:nth-child : 구조상 n번째 자식인 선택자
