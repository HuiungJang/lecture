position 속성

태그를 원하는 위치에 배치하는 속성값
default 값은 static이다.
position 속성은 위치를 지정하는 top,left,right,bottom 속성과 같이 사용한다.
position : static , relative, absoulte, fixed
relative : 부모의 position이 static일때 태그의 설정위치를 0,0으로 사용
absoulte : 부모의 position이 static일때 페이지 왼쪽 꼭지점을 0,0으로 사용
부모가 relative, absolute 포지션이면 부모의 0,0포지션이 시작위치다.
fixed : 특정태그를 지정위치에 고정해서 출력할 때 사용
