색상

css로 색상을 표시할 때는 생상 영문명, 16진수 RGB값,rgb(),rgba() 이용

투명도 있는 색상

여기도 content영역

background-color

태그마다 페이지에서 차지하는 각 영역이 있는데 그 영역에 대한 색상을 선택할 수 있게함
배경색을 지정하는 속성

  • html
  • css
  • javascript
  • jquery

background-clip

border-box : 박스 모델의 가작 외곽 테두리까지 적용
padding-box : 테두리 제외한 패딩범위까지 적용
content-box : 내용부분만 적용

border-box
padding-box
content-box

background-imgage

태그에 이미지를 넣을 떄 사용하는 속성
사진을 불러와서 출력 -> url(경로)

background-repeat

기본적으로 배경이미지를 설정하면 반복(바둑판식)으로 출력
조정할 수 있는 속성 ->repeat, repeat-x / repeat-y, no-repeat

background-size

배경이미지의 크기를 조정할 수 있는 속성
contain : 이미지가 영역안에 모두 들어갈 수 있게 확대, 축소 -> 이미지 모두 출력
cover : 영역이 꽉차게 설정함 -> 이미지 일부만 출력될 수 있음
숫자/백분율 : 크기를 지정

background-position

배경 이미지의 위치를 조정하는 속성
background-position : left || right || top || bottom || center || 숫자(절대, 상대)

background-origin

배경이미지가 차지하는 범위를 설정하는 속성
background-origin : border-box,padding-box,content-box

background-attachment

페이지가 스크롤에 의해 움직일때 배경 이미지의 위치를 설정하는 속성
scroll : 페이지가 움직이면 따라 움직임
fixed : 페이지가 이동해도 설정한 위치에 있음

linear-gradient

그라데이션 속성 background:linear-gradient(각도||방향, 시작색, 끝색)
