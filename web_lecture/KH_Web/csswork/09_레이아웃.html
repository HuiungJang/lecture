height/width

콘텐츠가 차지하고 있는 영역의 크기를 조절 할 수 있는 속성
상대크기와 절대크기가 있다.
상대크기 단위 -> 창크기가 줄면 창크기에 따라 변경
절대크기 단위 -> 창크기에 상관없이 고정된 위치에 나옴
height -> 높이 (세로)
width -> 넓이 (가로)
단위 : px -> 절대크기, % -> 상대크기
