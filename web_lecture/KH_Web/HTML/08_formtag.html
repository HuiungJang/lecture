form 태그

사용자가 입력한 data를 보내는 방식과 처리할 프로그램을 정하는 태그

주요 속성

method : 데이터를 어떻게 보낼꺼냐

  1. get : url 주소에 데이터를 추가해서 보낸다. 모든 데이터가 다 보여서 보안에 취약 -> 크기에 제한이 있음
  2. post : 패킷으로 보내는 방식. 데이터를 숨겨서 보냄-> 크기에 제한이 없음

action : 데이터를 어디서 처리할꺼냐 서버냐 페이지냐

post/get 방식으로 데이터 전송하기
