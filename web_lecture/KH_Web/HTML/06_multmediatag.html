이미지 태그

웹페이지에 사진이나 그림을 삽입할때 사용하는 태그
gif,jpg,png만 가져올 수 있다.
이미지 태그는 단일 태그로 속성으로 모든 것을 설정
src : 이미지가 있는 경로
height : 이미지 출력 높이(픽셀)
weight : 이미지 출력 넓이
alt : 이미지가 없을 때 나오는 설명문구

이미지 맵

이미지를 구역별로 나누어 링크를 걸어주는 태그로 img태그에 usemap속성을 지정해야함
map태그, area태그를 이용해서 이미지의 특정 부분을 클릭하면 링크로 이동하게 하는 태그
